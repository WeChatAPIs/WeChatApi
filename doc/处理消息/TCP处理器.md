#### 简要描述

- 添加TCP消息处理器，您需要自己实现TCP Server。
- 该类处理器使用短连接，发送完成后会自动断开。
- 消息的前四个字节是文本长度，可以先读四个字节再读取指定长度的文本。

#### 请求URL

- ` http://127.0.0.1:8888/api/`

#### 请求方式

- POST

#### 参数

| 参数名      | 必选 | 类型     | 说明   |   
|:---------|:---|:-------|------|   
| type     | 是  | int    | 接口编号 |   
| protocol | 是  | int    | 协议类型 |   
| host     | 是  | string | IP地址 |   
| port     | 是  | string | 端口号  |   

#### 请求示例

```
 {
  "type": 1001,
  "protocol": 1,
  "host": "127.0.0.1",
  "port": 18000
 }

```

自行实现，或使用http处理器
