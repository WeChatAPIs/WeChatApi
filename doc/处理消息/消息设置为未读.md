#### 简要描述

- 将指定用户的消息设置为已读，此功能会影响移动端。

#### 请求URL

- ` http://127.0.0.1:8888/api/`

#### 请求方式

- POST

#### 参数

| 参数名      | 必选 | 类型     | 说明     |   
|:---------|:---|:-------|--------|   
| type     | 是  | int    | 接口编号   |   
| userName | 是  | string | 用户wxid |   

#### 请求示例

```
 {
  "type": 10102,
  "userName": "wxid_xxx"
 }
```

#### 返回示例

``` 
  {
    "data": {
      "status": 0,
      "desc": "",
      "cookie": ""
    }
  }
```

#### 返回参数说明

| 参数名         | 类型     | 说明   |   
|:------------|:-------|------|   
| error_code  | int    | 错误代码 |   
| description | string | 错误描述 |   
| data        | json   | 业务数据 |   

#### 备注

- 更多返回错误代码请看[错误代码.md](../错误代码.md)
