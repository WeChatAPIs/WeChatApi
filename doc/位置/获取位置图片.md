##### 简要描述

- 根据经纬度，获取定位图片。

##### 请求URL
- ` http://127.0.0.1:8888/api/`
  
##### 请求方式
- POST 

##### 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|type |是  |int | 接口编号    |
|longitude |是  |float | 经度    |
|latitude |是  |float | 纬度    |
|width |否  |int | 图片宽    |
|height |否  |int | 图片高    |

##### 请求示例

```
 {
  "type": 27,
  "longitude": 104.00005,
  "latitude": 38.00005,
  "width": 425,
  "height": 162,
 } 
```

##### 返回示例 

``` 
  {
    "error_code": 10000,
    "description": "",
    "data": {
      "status": 0,
      "desc": "",
      "data": []
    }
  }
```

##### 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|error_code |int   |错误代码  |
|description|string|错误描述|
|data|json|业务数据|
