##### 简要描述

- 批量获取用户信息

##### 请求URL
- ` http://127.0.0.1:8888/api/`
  
##### 请求方式
- POST 

##### 参数

| 参数名       | 必选 | 类型   | 说明                     |   
|:----------|:---|:-----|------------------------|   
| type      | 是  | int  | 接口编号                   |   
| userNames | 是  | list | 用户wxid列表，单次请求数量不要超过100 |   

##### 请求示例

```
 {
  "type": 21,
  "userNames": ["wxid_xxxx","gh_xxxx","xxxx@chatroom"]
 } 
```

##### 返回参数说明 

| 参数名         | 类型     | 说明   |   
|:------------|:-------|------|   
| error_code  | int    | 错误代码 |   
| description | string | 错误描述 |   
| data        | json   | 业务数据 |
