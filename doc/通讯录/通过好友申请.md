##### 简要描述

- 通过好友申请

##### 请求URL
- ` http://127.0.0.1:8888/api/`
  
##### 请求方式
- POST 

##### 参数

| 参数名             | 必选 | 类型     | 说明               |
|:----------------|:---|:-------|------------------|
| type            | 是  | int    | 接口编号             |
| encryptUserName | 是  | string | 用户v3数据           |
| ticket          | 是  | string | 用户v4数据           |
| scene           | 否  | int    | 好友来源，可以从消息xml中获取 |

##### 请求示例

```
 {
  "type": 10035,
  "encryptUserName": "v3_xxxx",
  "ticket": "v4_xxxx",
 } 
```

##### 返回示例 

``` 
  {
    "error_code": 10000,
    "description": "",
    "data": {
      "status": 0,
      "desc": "",
    }
  }
```

##### 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|error_code |int   |错误代码  |
|description|string|错误描述|
|data|json|业务数据|

##### 备注 

- 更多返回错误代码请看首页的错误代码描述
