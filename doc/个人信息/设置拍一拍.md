##### 简要描述

- 设置别人拍一拍自己时显示的内容。

##### 请求URL
- ` http://127.0.0.1:8888/api/`
  
##### 请求方式
- POST 

##### 参数

| 参数名     | 必选 | 类型     | 说明    |   
|:--------|:---|:-------|-------|   
| type    | 是  | int    | 接口编号  |   
| content | 是  | string | 拍一拍内容 |   

##### 请求示例

```
 {
  "type": 10047,
  "content": "说：请你吃饭",
 } 
```

##### 返回示例 

``` 
  {
    "data": {
      "status": 0,
      "desc": "",
    }
  }
```

##### 返回参数说明 

| 参数名         | 类型     | 说明   |   
|:------------|:-------|------|   
| error_code  | int    | 错误代码 |   
| description | string | 错误描述 |   
| data        | json   | 业务数据 |
