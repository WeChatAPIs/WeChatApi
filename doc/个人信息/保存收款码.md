##### 简要描述

- 获取长效收款码。

##### 请求URL
- ` http://127.0.0.1:8888/api/`
  
##### 请求方式
- POST 

##### 参数

| 参数名    | 必选 | 类型  | 说明    |   
|:-------|:---|:----|-------|   
| type   | 是  | int | 接口编号  |   
| amount | 否  | int | 金额（分） |   

##### 请求示例

```
 {
  "type": 10087,
  "amount": 100,
 } 
```

##### 返回示例 

``` 
  {
    "data": {
      "status": 0,
      "desc": "",
        "data": {
            "partialName": "*名",
            "qrcodeUrl": "url" # 二维码url，可转成二维码供用户扫描
        },
    }
  }
```

##### 返回参数说明 

| 参数名         | 类型     | 说明   |   
|:------------|:-------|------|   
| error_code  | int    | 错误代码 |   
| description | string | 错误描述 |   
| data        | json   | 业务数据 |
