
#### 简要描述

- 获取登录二维码，如果目标客户端已经登录，则无法获取。

#### 请求URL
- ` http://127.0.0.1:8888/api/`
  
#### 请求方式
- POST 

#### 参数

| 参数名  | 必选 | 类型  | 说明   |   
|:-----|:---|:----|------|   
| type | 是  | int | 接口编号 |   

#### 请求示例

```
 {
  "type": 0
 } 
```

#### 返回示例 

``` 
  {
    "data": {
      "status": 0,
      "desc": "",
      "data": [] #返回二维码的字节数组，需自行转为图片。
    }
  }
```

#### 返回参数说明 

| 参数名         | 类型     | 说明   |   
|:------------|:-------|------|   
| error_code  | int    | 错误代码 |   
| description | string | 错误描述 |   
| data        | json   | 业务数据 |   

#### 备注 

- 更多返回错误代码请看[错误代码.md](../错误代码.md)
