#### 简要描述

- 检测链接是否被封禁。

#### 请求URL

- ` http://127.0.0.1:8888/api/`

#### 请求方式

- POST

#### 参数

| 参数名  | 必选 | 类型     | 说明   |   
|:-----|:---|:-------|------|   
| type | 是  | int    | 接口编号 |   
| url  | 是  | string | 链接   |   

#### 请求示例

```
{
  "type": 10111,
  "url": "https://xxx.com/xxxxxxxx"
} 
```

#### 返回示例

``` 
  {
    "data": {
      "status": 0,
      "desc": "",
      "data": []
    }
  }
```

#### 返回参数说明

| 参数名         | 类型     | 说明   |   
|:------------|:-------|------|   
| error_code  | int    | 错误代码 |   
| description | string | 错误描述 |   
| data        | json   | 业务数据 |   

#### 备注

- 更多返回错误代码请看[错误代码.md](../错误代码.md)
