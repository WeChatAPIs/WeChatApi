
#### 简要描述

- 上传朋友圈图片。

#### 请求URL
- ` http://127.0.0.1:8888/api/`
  
#### 请求方式
- POST 

#### 参数

| 参数名      | 必选 | 类型     | 说明     |   
|:---------|:---|:-------|--------|   
| type     | 是  | int    | 接口编号   |   
| filePath | 是  | string | 文件绝对路径 |   

#### 请求示例

```
 {
  "type": 5,
  "filePath": "C:\\xxx.jpg"
 } 
```

#### 返回示例 

``` 
  {
    "data": {
      "status": 0,
      "desc": "",
    }
  }
```

#### 返回参数说明 

| 参数名         | 类型     | 说明   |   
|:------------|:-------|------|   
| error_code  | int    | 错误代码 |   
| description | string | 错误描述 |   
| data        | json   | 业务数据 |
