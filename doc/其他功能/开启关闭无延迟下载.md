##### 简要描述

- 开启/关闭无延迟下载
- 微信在指定时间段，会将cdn消息放置到延迟下载队列（需要手动触发下载），可使用此接口开启或关闭无延迟下载。

##### 请求URL

- ` http://127.0.0.1:8888/api/`

##### 请求方式

- POST

##### 参数

| 参数名     | 必选 | 类型   | 说明      |   
|:--------|:---|:-----|---------|   
| type    | 是  | int  | 接口编号    |   
| bEnable | 是  | bool | 是否启用防撤回 |   

##### 请求示例

```
  {
  "type": 10094,
  "bEnable": true,
 }
```

##### 返回示例

``` 
  {
    "data": {
      "status": 0,
      "desc": "",
      "data": []
    }
  }
```

##### 返回参数说明

| 参数名         | 类型     | 说明   |   
|:------------|:-------|------|   
| error_code  | int    | 错误代码 |   
| description | string | 错误描述 |   
| data        | json   | 业务数据 |   

##### 备注

- 更多返回错误代码请看[错误码](../错误代码.md)
