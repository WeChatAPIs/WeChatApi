##### 简要描述

- 邀请好友加入群聊

##### 请求URL
- ` http://127.0.0.1:8888/api/`
  
##### 请求方式
- POST 

##### 参数

| 参数名              | 必选 | 类型     | 说明           |   
|:-----------------|:---|:-------|--------------|   
| type             | 是  | int    | 接口编号         |   
| chatroomUserName | 是  | string | 群聊wxid       |   
| userNames        | 是  | list   | 待邀请wxid列表    |   
| accessMessage    | 否  | string | 用于向管理员发送验证信息 |   

##### 请求示例

```
 {
  "type": 32,
  "chatroomUserName": "xxx@chatroom",
  "userNames": ["wxid_xxxx1","wxid_xxxx2"],
 } 
```

##### 返回示例 

``` 
  {
    "error_code": 10000,
    "description": "",
    "data": {
      "status": 0,
      "desc": "",
    }
  }
```

##### 返回参数说明 

| 参数名         | 类型     | 说明   |   
|:------------|:-------|------|   
| error_code  | int    | 错误代码 |   
| description | string | 错误描述 |   
| data        | json   | 业务数据 |
