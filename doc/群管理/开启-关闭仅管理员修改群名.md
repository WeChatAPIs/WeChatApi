##### 简要描述

- 开启或关闭仅管理员可修改群聊名称

##### 请求URL
- ` http://127.0.0.1:8888/api/`
  
##### 请求方式
- POST 

##### 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|type |是  |int | 接口编号    |
|chatroomUserName |是  |string | 群聊wxid    |
|isOnlyAdminChangeName     |是  |bool | 是否仅管理员可修改群名    |

##### 请求示例

```
 {
  "type": 10031,
  "chatroomUserName": "xxx@chatroom",
  "isOnlyAdminChangeName": true,
 } 
```

##### 返回示例 

``` 
  {
    "error_code": 10000,
    "description": "",
    "data": {
      "status": 0,
      "desc": "",
    }
  }
```

##### 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|error_code |int   |错误代码  |
|description|string|错误描述|
|data|json|业务数据|
